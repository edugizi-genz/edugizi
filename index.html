Keamanan Admin Guru SMAN 1 Parepare: Akses khusus rekapitulasi data guru dilindungi sandi via enkapsulasi dialog box. Sandi default yang diset: guru123 atau admingizi. Anda juga dapat membukanya dengan melakukan klik sebanyak 5 kali secara cepat pada logo EDUGIZI di navbar atas. bemana kalo mau langsung kasih masuk datanya di goggle sppet Untuk memasukkan data hasil ujian siswa secara otomatis ke Google Sheets, Anda tidak bisa hanya menggunakan kode JavaScript biasa (localStorage) karena data tersebut hanya tersimpan di komputer masing-masing siswa. Solusi paling mudah dan gratis yang sering digunakan oleh guru adalah menghubungkan kode HTML/JavaScript Anda dengan Google Apps Script (Web App). Berikut adalah langkah-langkah lengkap dan perubahan kode yang perlu Anda lakukan: Langkah 1: Buat Google Sheets & Apps Script Buka Google Sheets, buat dokumen baru, dan beri nama (misalnya: Data Nilai EduGizi SMAN 1 Parepare). Buat baris pertama (Header) di spreadsheet Anda dengan urutan berikut: Kolom A: Nama Siswa Kolom B: Kelas Kolom C: Skor Pre-Test Kolom D: Skor Post-Test Kolom E: Waktu Ujian Di menu atas Google Sheets, klik Ekstensi > Apps Script. Hapus semua kode yang ada di sana, lalu tempelkan kode Apps Script berikut: JavaScript function doPost(e) { try { // Mengambil data yang dikirim dari aplikasi web HTML var data = JSON.parse(e.postData.contents); // Membuka spreadsheet aktif var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); // Memasukkan data ke baris paling bawah yang kosong sheet.appendRow([ data.nama, data.kelas, data.pretest, data.posttest, data.waktu ]); // Memberikan respon sukses balik ke HTML return ContentService.createTextOutput(JSON.stringify({"status": "success"})) .setMimeType(ContentService.MimeType.JSON); } catch(error) { return ContentService.createTextOutput(JSON.stringify({"status": "error", "message": error.toString()})) .setMimeType(ContentService.MimeType.JSON); } } Klik tombol Terapkan (Deploy) di kanan atas > pilih Terapkan Baru (New Deployment). Klik ikon gerigi (Jenis penerapan) > pilih Aplikasi Web (Web App). Setel pengaturannya seperti ini: Deskripsi: Kirim data EduGizi. Jalankan sebagai (Execute as): Saya (Email Anda sendiri). Yang memiliki akses (Who has access): Siapa saja (Anyone). (Ini penting agar murid bisa mengirim data tanpa perlu login akun Google mereka). Klik Terapkan (Deploy). Jika meminta izin akses, klik Sediakan Akses / Allow. Salin URL Aplikasi Web yang diberikan (Ujung URL-nya biasanya berupa /exec). Langkah 2: Ubah Kode JavaScript di HTML Anda Sekarang, kembali ke file HTML Anda. Cari fungsi selesaiPengerjaanKuis() di bagian bawah
